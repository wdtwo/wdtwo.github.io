1.准备工作

  1. 安装express
    • npm install express -g //全局安装express
    • npm install express-generator -g //安装全局变量
  2. 初始化项目
    • cd example //进入项目文件夹
    • express project //创建项目
  3. 执行如下命令
    • cd project //进入目录
    • npm install //安装依赖
  • /bin:用来启动应用（服务器）
  • /public: 存放静态资源目录
  • /routes：路由用于确定应用程序如何响应对特定端点的客户机请求
  • /views: 模板文件所在目录 文件格式为.jade
  • 目录app.js程序main文件 这个是服务器启动的入口
