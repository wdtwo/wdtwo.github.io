什么是图表形态？

图表形态是一组自动搜索图表上重复价格形态的指标。图表形态帮助交易者建立价格变动的概率预测并确定目标水平。形态可分为3种类型，具体取决于形成后价格预期移动的方向：

  • 延续形态 : 延续形态包括旗帜和三角旗等图案。
  • 反转形态 : 反转形态有双顶和三顶、头肩顶、楔形。
  • 不确定形态 : 不确定形态是三角形和矩形。

除了代表各种几何形状的形态外，还有基于各种理论的形态，例如艾略特波浪理论。 根据这一理论，价格变动是周期性的，会经历几个阶段。了解当前价格处于哪个阶段后，就可以预测其进一步走势。
您可以在技术板块形态组的指标菜单中找到所有可用的内置图表形态。
使用指标时，如果找到相应的形态，它们将显示在图表上。
您可以在相关文章中获得有关自动检测图表形态的内置指标的更多详细信息。

请注意，自动图表形态提供的有关预期价格目标的信息，并不是对您个人应该做什么的建议。请勿将此数据作为投资建议。它只能用于教育和研究。与任何交易一样，请始终谋定而后动。

如果指标发现两个相交形态，则优先考虑状态为等待的形态。 如果相交形态的状态为“失败”或“已达到”或两者的状态均为“等待”，则图表将显示底部价格值最相似的形态，即更准确。如果状态为”无法定义“的形态与具有不同状态的形态相交，则该形态将被删除。

图表形态三底


三底是由三个连续的同一水平低点（允许价格值略有差异）和它们之间的两个中间高点形成的反转形态。预计在价格触及第三个低点然后转身并超过中间最大值水平后，价格将继续进一步上涨，幅度约为高点和低点之间的差值。

该指标搜索最近 600 根K线的形态。 该形态由指示价格走势的线(Price Line)和颈线(Neck Line)组成。 颈线沿着中间最大值的最高点绘制。除了最后一条(点7)外，每条价格线的起点和终点都位于 5/5 轴心。点 7 位于最后一条价格线与颈线的交点。在 进行中 模式下，指标不仅寻找已形成的形态，还寻找新兴的形态。这种模式的第三个底部可能不在枢轴中，形成它的价格线将被点缀。

当第三个底部出现后，价格（收盘价）升至颈线上方时，该形态被视为已形成。以目标标签结尾的水平虚线表示形态形成后的预期价格水平。随着新K线的出现，该线向右延伸直到最后一个K线，直到形态状态从等待变为另一个。标签工具提示显示形态的价格和当前状态。 当状态改变时，标签的颜色也会改变。 一共有4种状态：

  • 等待 : 价格没有达到预期水平，也没有跌破最后的底部。
  • 到达 : 价格达到预期水平。
  • 失败 : 价格没有达到预期水平，跌破了最后一个底部。
  • 无法定义 : 该指标无法明确确定形态的状态。

警报：

  • 新形态 : 当图表上出现新形态时会触发它。如果某个形态的第一底部或第二底部或中间最大值之一的位置不同，则该形态被视为新形态。
  • 形态形成 : 当第三个底部出现后，出现一条收盘价高于颈线的K线时，就会触发该交易。
  • 形态失败 : 当形态状态变为失败时触发。仅当形态完全形成时触发。
  • 目标已达到 : 当价格达到形态目标时触发。仅当形态完全形成时触发。

输入：

  • 形态 : 定义将根据其状态绘制的形态。可能的值：全部、等待和已到达、仅等待、最后等待。 如果选择最后等待 ，则只会显示一个状态为等待的形态，其第一个点位于所有其他形态的右侧。
  • 价格目标 : 定义将根据形态的状态绘制哪些价格目标。 可能的值：全部、仅等待、等待和已达到、无。
  • 趋势高度 : 设置相对于模式本身高度的模式前趋势所需的高度。形态的高度是最低底部的价格与中间最高点的最高价格之差。百分比值。
  • 进行中 : 搜索尚未完全形成的形态。
  • 底部 : 底部深度的最大允许差异。底部的深度是根据中间最大值中的最高值计算的。百分比值。
  • 中间最大值 : 中间最大值的最大允许高度差。 最大值的高度是从底部的最低值开始计算的。百分比值。

图表形态三角形


三角形是一种不确定的形态，可以预示价格上涨和下跌。目标价格水平取决于价格突破该形态的方向。预计在形态突破后，价格将在突破方向上大致达到三角形底部的高度。

该指标搜索最近 600 根K线的形态。该形态由表示价格变动的线(价格线)和形成三角形(Triangle)的线组成。每条价格线的起点和终点位于 5/5 轴心点。形态的突破由收盘价决定。 不允许三角形的线与点 1 和点 5 之间的区间中的接近值相交。在进行中模式下，指标不仅寻找已形成的形态，还寻找新兴的形态。这种形态的最后两点可能不在枢轴上，最后一条价格线将被点缀。

以目标标签结尾的水平虚线表示形态完成后的预期价格水平。如果形态未被打破，则指标将绘制 2 条这样的线 – 每个方向各一条。突破后，只剩下突破方向的线。随着新K线的出现，该线向右延伸直到最后一根K线，直到形态状态从等待变为另一个。标签工具提示显示形态的价格和当前状态。当状态改变时，标签的颜色也会改变。一共有4种状态：

  • 等待 : 三角形不破或突破，价格在向下突破的方向上还没有达到目标位，也没有超越三角形的最后一个对立点。
  • 到达 : 价格突破其中一条三角形线并在同一方向达到目标。
  • 失败 : 价格突破了三角形的一条线，并且在没有达到目标的情况下，超过了与击穿方向相反的最后一点。
  • 无法定义 : 该指标无法明确确定形态的状态。

警报：

  • 新形态 : 当图表上出现新形态时会触发它。如果某个形态改变了点 1、2 或 4 的位置，则该形态被视为新形态。
  • 形态突破 : 当出现在三角形之外收盘的K线时，就会触发它。如果形态最后一个点所在的同一条线发生突破，则在尝试重建点 5 失败的情况下，如果自突破以来出现的K线数不超过 5 个，则在反向重建时将触发此警报 。
  • 形态失败 : 当形态状态变为失败时触发。仅当形态完全形成时触发。
  • 目标已达到 : 当价格达到形态目标时触发。仅当形态完全形成时触发。

输入：

  • 形态 : 定义将根据其状态绘制的形态。可能的值：全部、等待和已到达、仅等待、最后等待。如果选择最后等待，则只会显示一个状态为等待的形态，其第一个点位于所有其他形态的右侧。
  • 价格目标 : 定义将根据形态的状态绘制哪些价格目标。可能的值：全部、仅等待、等待和已达到、无。
  • 允许偏差 : 点 3 与相应直线的最大允许偏差。它计算为点 3 所在的同一根K线上的三角形线之间距离的百分比。
  • 进行中 : 寻找新出现的形态。

跳空（Gaps）

定义

跳空是图表上两个K线之间的价格发生显着变化而它们之间没有发生任何交易的区域。当市场通常关闭的时间发生强烈的情绪转变时，通常会出现这种差距。该指标在图表上突出显示这些差距并将它们进一步扩展，直到它们被覆盖（即，当新的K线穿过该跳空时）。
指标突出显示的跳空有两个主要变化：如果当前K线的最低价大于前一根K线的最高价，则为向上缺口，默认情况下将显示为绿色。 如果当前K线的最高价低于前一根K线的最低价，则这种跳空（向下跳空）将默认显示为红色。如果在当前商品上没有找到符合指定设置的跳空，将显示相应的消息。
默认情况下，指标根据其宽度过滤跳空，并且仅在跳空至少为最后14根K线的平均K线范围（高 - 低）的30%时显示跳空。可以在输入部分更改偏差%。
此外，该指标无限期地扩大跳空，直到它们被新的价格走势所覆盖。这也可以通过启用最大跳空轨迹长度Max Gap Trail Length限制在设置中更改。启用后，如果跳空超出指定的K线数（默认为 300），则K线将被关闭。

总结

跳空标志着在交易所不活跃期间价格发生急剧变化的时期。交易者可以利用这些信息从当前价格突然变化的情况中获利。

更改关闭差距所依据的逻辑。 默认情况下，此设置处于关闭状态，一旦柱线进入跳空，跳空就会完全闭合，即使该K线仅触及t跳空的一小部分。 当设置打开时，每当一根K线进入缺口区域但没有完全覆盖它时，跳空部分关闭，并且没有被价格运动覆盖的部分进一步扩展（并且可以被未来的K线覆盖） .
