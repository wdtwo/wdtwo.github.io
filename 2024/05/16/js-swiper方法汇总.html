跳转到指定索引

在Loop模式下Swiper切换到指定slide。切换到的是slide索引是realIndex
比如你的swiper是loop模式且只复制一个slide(默认情况)，你想跳转到第三个slide时（activeIndex:3/realIndex:2），应该使用slideTo(3)或者slideToLoop(2)。

禁止触摸滑动

在官方文档的Touches(触发条件)中
设置是否允许触摸滑动。如果设为 false 时，那么切换幻灯片的唯一方法是使用外部 API 函数，例如slideNext()或slidePrev()或slideTo() 等。等同于Swiper3.x 的 onlyExternal。

手动操作以后不停止轮播
